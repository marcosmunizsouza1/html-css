Listas Ordenadas

Para criarmos lista ordenada, usamos a tag OL.

Dentro da lista ordenada, podemos ordenar usando a tag LI.

Existe uma configurações para mudar a o tipo de lista (1, A, a, I, i) mudando o type e iniciando em um valor predeterminado usando o start.

    - type = 1 (numérico)

  1. Acordar;
  2. Ligar para o João;
  3. Tomar Café;
  4. Escovar os dentes;
  5. Ir para a Faculdade;

    - type = A (Letra Maiúscula)

  1. Acordar;
  2. Ligar para o João;
  3. Tomar Café;
  4. Escovar os dentes;
  5. Ir para a Faculdade;

    - type = a (Letra Minúscula)

  1. Acordar;
  2. Ligar para o João;
  3. Tomar Café;
  4. Escovar os dentes;
  5. Ir para a Faculdade;

    - type = I (Algarismo Romando Maiúsculo)

  1. Acordar;
  2. Ligar para o João;
  3. Tomar Café;
  4. Escovar os dentes;
  5. Ir para a Faculdade;

    - type = i (Algarismo Romando Minúsculo)

  1. Acordar;
  2. Ligar para o João;
  3. Tomar Café;
  4. Escovar os dentes;
  5. Ir para a Faculdade;

    - start = ñ (número que iniciará a listagem)

  1. Acordar;
  2. Ligar para o João;
  3. Tomar Café;
