Head Contém metadados da página, ou seja, informações que não aparecem diretamente para o usuário.
  • <title> → título exibido na aba do navegador
  • <meta charset="UTF-8"> → define a codificação de caracteres
  • <meta name="viewport"> → responsividade em dispositivos móveis
  • <link rel="stylesheet"> → ligação com arquivos CSS
  • <script> → inclusão de scripts JavaScript
  • <meta name="description"> → descrição da página para SEO
  • <meta name="author"> → autor da página
Main Área principal do conteúdo visível da página. Aqui ficam os elementos que o usuário realmente vê.
